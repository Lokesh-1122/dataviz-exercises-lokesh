# Data Visualization Exercises

## Overview
This repository contains weekly Data Visualization exercises developed using Python. The project demonstrates various visualization techniques and data analysis methods using real-world datasets.

## Features
- Bar Charts
- Line Charts
- Scatter & Bubble Charts
- Hierarchical Visualizations
- Choropleth Maps
- Interactive Dashboards with Streamlit

## Technologies Used
- Python
- Pandas
- Plotly Express
- Streamlit
- Jupyter Notebook

## Folder Structure
```
Week02 - Bar Charts
Week03 - Multi-Series Line Charts
Week04 - Scatter & Bubble Charts
Week05 - Data Visualization
Week06 - Hierarchical Visualizations
Week07 - Visualization Exercises
Week08 - Choropleth Maps
Week09 - Data Visualization Notebook
Week10 - Streamlit Dashboard
```

## Learning Outcomes
- Data preprocessing and analysis
- Exploratory Data Analysis (EDA)
- Interactive data visualization
- Geographic data visualization
- Dashboard development
- Data storytelling

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/your-username/dataviz-exercises.git
```

2. Install the required libraries:
```bash
pip install pandas plotly streamlit
```

3. Open the Jupyter notebooks to explore the weekly exercises.

4. Run the Streamlit dashboard:
```bash
streamlit run Lecture10_exerciseAnswer.py
```
